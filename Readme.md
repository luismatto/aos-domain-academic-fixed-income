# AOS Domain Academic Fixed Income

Domain repository for the module Valoracion de Activos Financieros: Renta Fija.

## Status

FI-DER-0001 Domain Repository Bootstrap in progress.

## Architecture

This repository follows AOS Object-Centric Architecture AOCA-001.

Knowledge is represented as Academic Objects. Policies and asset generation rules are applied to objects, not to chapters or files.
