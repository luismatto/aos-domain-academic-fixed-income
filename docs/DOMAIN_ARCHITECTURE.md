# Domain Architecture

## Principle

This repository is organized around Academic Objects.

## AOCA-001

All knowledge in the AOS ecosystem is represented as an Academic Object.

Domain repositories are not organized around chapters, sessions or learning units. They are organized around self contained academic objects.

Pedagogy, assessment and assets are transformations applied to those objects.
